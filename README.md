## 🚀 How to Deploy on GitHub Pages (Free Hosting)

1. Create a new GitHub repository named **`portfolio`**
2. Upload `index.html` and `README.md` to the repo
3. Go to **Settings → Pages**
4. Under **Source**, select `main` branch → `/ (root)` → Save
5. Wait ~1 minute — your site will be live at `https://YOUR_USERNAME.github.io/portfolio`
